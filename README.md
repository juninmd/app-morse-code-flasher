# Morse Code Flasher

A React Native (Expo) application that translates text to Morse Code and flashes the device's flashlight accordingly.

## Features
- Text to Morse conversion
- Visual display of Morse code
- **Real Flashlight Transmission** (Uses device Torch)

## Development

### Prerequisites
- Node.js
- pnpm

### Setup
```bash
pnpm install
```

### Running
```bash
pnpm start
```

### Testing
```bash
pnpm test
```
